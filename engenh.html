    Resumo

    Pronto Zé, essenicalmente esta componente vai servir como se fosse um "mini" arcgis, só que 1000x mais facil de se usar, com exportaçao em shape e Geojson. Neste momento estou a trabalhar na parte de análise espacial, adicionar mais ferramentas, para já só tem buffer.

    Passo 1 de 3

    Como queres trabalhar?

    Escolhe se cada geometria desenhada vai ter apenas a forma, ou se vai ter atributos associados (como no QGIS/ArcGIS).

    Simples Apenas desenhar geometrias, sem atributos.
    Com atributos Define até 3 campos, com classes predefinidas se quiseres.
    Passo 2 de 3

    Define os atributos

    Até 3 atributos. Para categorias (campo "Categórico"), define as classes e respetiva cor — tal como uma simbologia categorizada no QGIS.

    Passo 2 de 2

    Que tipo de geometria?

    Um shapefile só pode conter um tipo de geometria — escolhe o que vais desenhar nesta sessão.
